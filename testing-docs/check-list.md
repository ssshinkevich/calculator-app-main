# Чек-лист тестирования : Web-Calculator-app

## Визуальное тестирование UI и состояние по умолчанию

### Общее
- Приложение корректно отображается на Desktop ( 1600 x 900 )
- Приложение корректно отображается на Mobile ( 430> )
- Значок Favicon отображается 
- Заголовок вкладки браузера соответствует названию

### Элементы управления
- По умолчанию на дисплеее отображаетя 0
- Переключатель темы имеет 3 позиции
- Тема 1: Цветовая палитра ( кнопки, текст, фон ) соответствуют макету
- Тема 2: При переключении цвета меняются на светлую тему согласно макету
- Тема 3: При переключении цвета меняются на фиолетовую тему согласно макету
- При наведении курсора ( hover ) на кнопки они меняют яркость/цвет
- При нажатии на кнопки они меняют яркость/цвет
- Скругления углов элементов управления соответствуют макету
- Текст элементов управления отцентрирован по вертикали и горизонтали

## Функциональное тестирование

### Арифметические функции ( Happy path )
- Сложение двух целых чисел ( 2 + 1 )
- Вычитание двух целых чисел ( 8 - 5 )
- Умножение двух целых чисел ( 5 * 7 )
- Деление двух целых чисел ( 12 / 3 )
- Сложение двух дробных чисел ( 0.1 + 2.5 )
- Вычитание двух дробных чисел ( 5.5 - 0.4 )
- Умножение двух дробных чисел ( 5.1 * 7.3 )
- Деление двух дробных чисел ( 12.9 / 3.3 )
- Получени отрицательного результата ( 10 - 100 )

### Сложная логика приложения
- Приоритет операций ( 3 + 5 * 2 ) должно быть 13
- Цепочка вычислений без нажатия "равно" ( 3 + 5 + 2 = )
- Продолжение вычислений с результатом ( 3 + 5 = 8, затем + 2 = 10 )

### Кнопки управления
- Кнопка RESET сбрасывает дисплей в 0 и очищает память операций
- Кнопка DEL удаляет только последний введенный символ
- Кнопка DEL ничего не делает если на экране только 0

## Валидация и негативные сценарии
- Деление на 0 ( Проверка сообщения об ошибке или поаведения, например Infinity )
- Ввод нескольких точек в одном числе ( 1.2.1 должно блокироваться или игнорироваться )
- Ввод точки перед числом ( .7 должно обрабатываться как 0.7 )
- Ввод максимально допустимого числа ( 15 знаков. Проверка на переполнение экрана )
