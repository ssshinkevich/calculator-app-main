# Тест-план : Web-Calculator-app

## 1. Введение
Данный документ описывает стратегию и план тестирования веб-приложения "Calculator"
**Цель** : Обеспечить стабильность работы интерфейса, корретное вычисление математических операций, валидацию вводимых данных.

## 2. Объекты тестирования
### 2.1. Функционал, входящий в тестирование
1. **Основные математические операции** : Сложение, вычитание, умножение, деление
2. **Логика вычислений** : Приоритет операций, работа с целыми и дробными числами, работа с отрицательными числами
3. **Управление состоянием** : Очистка ввода ( RESET ), удаление последнего символа ( DEL )
4. **Валидация данных** : Обработка деления на ноль, блокировка ввода некорректных символов ( Двойная точка, буквы ), ограничение максимальной длины ввода.
5. **UI / UX** : Адаптивность ( Desktop / mobile ), визуальный отклик кнопок, работоспособность смены цветовой темы
6. **Доступность** : Поддержка ввода с клавиатуры и совместимость с экранными дикторами

### 2.2. Функционал, не входящий в тестирование
- Нагрузочное тестирование
- Тестирование безопасности ( Приложение не имеет backend части )
- Локализация ( Приложение поддерживает только арабские цифры )

## 3. Стратегия тестирования
Для обеспечения полного покрытия тестами будут использоваться **Техники тест-дизайна**:
1. Классы эквивалентности:
- Положительные числа ( 1, 100 )
- Отрицательные числа ( -1, -100 )
- Дробные числа ( 0.5, 1.99 )
- Ноль ( 0 )
2. Граничные значения:
- Максимально допустимое число символов ( 15 )
- Минимально возможное число
3. Предугадывание ошибок:
- Деление на ноль
- Ввод нескольких математических операторов подряд ( 1 ++ 1 / 5 -- 5)
- Ввод точки перед числом ( .1 )

## 5. Критерии начала и завершения
### Критерии начала
- Код приложения доступен в репозитории GitHub
- Приложение запускается локально без критических ошибок блокирующих запуск
- Утверждены требования к функционалу
#### Критерии завершения
- Все тест кейсы со статусами *high* и *critical* успешно пройдены
- Отсутствие открытых багов с приоритетом *high* и *critical*
- Все найденные баги задокументированы
