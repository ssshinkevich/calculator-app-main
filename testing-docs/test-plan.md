# Тест-план : веб - калькулятор

## 1. Введение
Проведение функционального и UI тестирование веб-приложения "Калькулятор" для проверки корректности математических вычислений.

## 2. Объекты тестирования
- Основные математические операции ( +, -, *, / )
- Логика работы с дробными числами ( Float )
- Обработка ошибок ( Деление на 0 )
- Интерфейс ( Адаптивность, реакция кнопок )

## 3. Окружение
- **OS** : MacOS Tahoe 26.0.1
- **Browsers** : Chrome, Safari, Firefox
- **Tools** : DevTools для проверки верстки

## 4. Виды тестирования
- **Позитивное** : стандартные вычисления
- **Негативное** : ввод некорректных данных, деление на 0
- **Граничные значения** : Ввод максимально длинного числа
